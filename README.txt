###### 使用说明 v0.1######

  本工具是通过编写Shell脚本，以SNMP方式定期（根据自身需求设定频率）获取华为交换机的ARP信息，并通过Python将数据切分存入Mysql数据库当中。通过SQL语句可以对当前网络中某个时间点存在的所有IP地址进行查询操作。

0x01:设计初衷

安全运维工作中，因为历史原因或者其他原因，网络中IP的分配情况信息不够准确，借助此工具辅助统计资产使用情况，便于系统的基础的安全加固工作开展并做细。


0x02:使用

赋予arp.sh和db.py可执行权限，并设定定时任务的执行频率，任务的执行结果可查看arp_log.txt和db_log.txt；
host.txt中填写网络中的汇聚层（有arp表）交换机的ip地址；

0x03:其他
OID.txt是交换机OID的参考数值；
createtable.sql 是Mysql的建库/表的SQL语句；

0x04:缺点

没有方便的前端查询展示界面，只能通过登录Mysql使用sql语句进行数据信息查询，计划下一版本添加一个人性化的前端界面。
